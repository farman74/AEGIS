# 00-Core

## Objectif

Contient les documents fondateurs d'AEGIS. Ce dossier définit l'identité du projet, ses valeurs fondamentales et ses standards non-négociables.

Tout agent IA doit lire ce dossier en premier, avant toute autre action.

## Structure

```
00-Core/
├── README.md               ← Ce fichier
├── PROJECT_CONSTITUTION.md ← La loi fondamentale
├── ENGINEERING_PRINCIPLES.md ← Les 20 principes obligatoires
├── QUALITY_STANDARD.md     ← Standards de qualité
└── AI_MANIFESTO.md         ← Règles de comportement IA
```

## Comment utiliser

1. Lire `PROJECT_CONSTITUTION.md` — c'est la loi.
2. Lire `ENGINEERING_PRINCIPLES.md` — ce sont les règles.
3. Lire `QUALITY_STANDARD.md` — c'est le barème.
4. Lire `AI_MANIFESTO.md` — c'est le code de conduite.

Ces documents ne changent jamais sans un ADR validé.
