# Mauvais Exemples — Frontend

Ce dossier contient des exemples de mauvaise implémentation frontend à éviter.

## Anti-patterns identifiés

- Clés API exposées côté client
- Logique métier dans les composants UI
- Appels Supabase directs sans service dédié
- Absence de gestion des erreurs
