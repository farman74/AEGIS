# Bons Exemples — Frontend

Ce dossier contient des exemples de bonne implémentation frontend pour ExamOS.

## Principes appliqués

- Composants atomiques et réutilisables
- Logique métier séparée des composants UI
- Connexion Supabase via le client officiel uniquement
- Gestion des erreurs affichée à l'utilisateur
- Aucune clé API exposée côté client
