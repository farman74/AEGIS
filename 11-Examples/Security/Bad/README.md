# Mauvais Exemples — Sécurité

Ce dossier contient des exemples de mauvaises pratiques sécurité à éviter.

## Anti-patterns identifiés

- Clés API en dur dans le code
- Absence de RLS
- Pas de vérification des permissions côté serveur
- Actions non journalisées
