# Bons Exemples — Backend

Ce dossier contient des exemples de bonne implémentation backend pour ExamOS.

## Principes appliqués

- Logique métier dans les Services, pas dans les Controllers
- Connexion à Supabase via le client officiel
- Validation des entrées avant traitement
- Erreurs gérées explicitement
- API Resources pour les réponses
