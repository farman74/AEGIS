# Mauvais Exemples — Backend

Ce dossier contient des exemples de mauvaise implémentation backend à éviter.

## Anti-patterns identifiés

- Logique métier dans les Controllers
- Utilisation de bases de données autres que Supabase
- Absence de validation des entrées
- Retour de modèles bruts
- Absence de gestion d'erreurs
