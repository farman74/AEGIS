# Mauvais Exemples — Database

Ce dossier contient des exemples de mauvaise gestion de base de données à éviter.

## Anti-patterns identifiés

- Utilisation de PostgreSQL local, SQLite, Firebase, MongoDB ou Prisma (✖ interdit)
- Tables sans contraintes
- Absence de politiques RLS
- Migrations irréversibles
- Absence d'index sur les clés étrangères
