# Bons Exemples — Database

Ce dossier contient des exemples de bonne gestion de base de données avec Supabase pour ExamOS.

## Principes appliqués

- Tables avec contraintes d'intégrité référentielle
- Politiques RLS définies sur toutes les tables sensibles
- Colonnes `created_at` et `updated_at` systématiques
- Index documentés et justifiés
- Migrations réversibles
