# Bons Exemples — Intelligence Artificielle

Ce dossier contient des exemples de bonne intégration IA pour ExamOS.

## Principes appliqués

- Sortie structurée avec score, confiance et commentaires
- Explicabilité : chaque note est justifiée par une règle du barème
- Reproductibilité : mêmes entrées → même résultat
- Versioning du prompt et du modèle enregistré
