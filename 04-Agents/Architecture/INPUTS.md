# Agent Architecture — Entrées

## Objectif

Définir les entrées acceptées par l'Agent Architecture.

## Entrées acceptables

- Demande de validation d'architecture
- Besoin de création d'un ADR
- Question sur une décision technique
- Demande d'analyse d'impact d'une modification
- Problème de cohérence architecturale identifié

## Format attendu

```
Contexte : [Description du problème ou de la demande]
Objectif : [Ce que l'on veut obtenir]
Contraintes : [Contraintes éventuelles]
```

## Documents à fournir

- Feature Contract validé si applicable (`10-Templates/Feature.md`)
- ADR existant si modification d'une décision passée
