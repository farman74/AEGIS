# Agent Documentation — Workflow

## Workflow

```
Identification des documents à mettre à jour
    ↓
Vérification de la cohérence avec nouvellesnormes.pdf
    ↓
Mise à jour des documents concernés
    ↓
Vérification de tous les liens internes
    ↓
Vérification absence de références cassées
    ↓
Validation de la checklist (09-Checklists/Documentation.md)
    ↓
Inclusion dans la PR du code associé
```
