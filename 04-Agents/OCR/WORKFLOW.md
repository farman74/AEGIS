# Agent OCR — Workflow

## Workflow

```
Réception de l'image
    ↓
Amélioration qualité
    ↓
Détection des zones
    ↓
Lecture du texte
    ↓
Structuration JSON
    ↓
Calcul du score de confiance
    ↓
Stockage via Supabase Storage
    ↓
Envoi au moteur de correction IA
```
