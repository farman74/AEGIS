# Agent OCR — Responsabilités

## Responsabilités

1. Recevoir les images de copies scannées
2. Améliorer la qualité de l'image
3. Détecter les zones de réponses
4. Lire le texte manuscrit
5. Structurer les réponses en JSON
6. Conserver les scores de confiance
7. Gérer les copies partiellement lisibles
8. Stocker les résultats via Supabase Storage
