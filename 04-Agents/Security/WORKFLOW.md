# Agent Security — Workflow

## Workflow

```
Réception d'une demande d'audit
    ↓
Consultation de 05-Engineering/Security/RULES.md
    ↓
Analyse de la fonctionnalité
    ↓
Vérification RLS, auth, permissions
    ↓
Vérification absence de secrets exposés
    ↓
Validation de la checklist (09-Checklists/Security.md)
    ↓
Rapport de sécurité
```
