# Agent Database — Responsabilités

## Responsabilités

1. Concevoir les schémas de tables Supabase
2. Rédiger et appliquer les migrations
3. Définir les politiques RLS sur toutes les tables sensibles
4. Optimiser les index et les performances
5. S'assurer de l'auditabilité de toutes les opérations
6. Documenter le schéma de données
7. S'assurer que toutes les migrations sont réversibles

## Ce que cet agent ne fait PAS

- N'utilise pas de base de données autre que Supabase
- Ne stocke pas de secrets dans la base de données
