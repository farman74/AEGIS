# Agent AI — Workflow

## Workflow

```
Réception question + réponse + barème
    ↓
Construction du prompt (10-Templates/Prompt.md)
    ↓
Appel du modèle IA
    ↓
Parsing de la réponse
    ↓
Vérification de l'explicabilité
    ↓
Stockage du résultat dans Supabase
    ↓
Enregistrement version prompt + modèle
```
