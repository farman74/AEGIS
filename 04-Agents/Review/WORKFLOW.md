# Agent Review — Workflow

## Workflow

```
Réception de la PR
    ↓
Lecture de la description et du Feature Contract
    ↓
Vérification des checklists (09-Checklists/Review.md)
    ↓
Analyse du code
    ↓
Vérification Supabase
    ↓
Vérification des tests
    ↓
Vérification de la documentation
    ↓
Décision : Approuver | Demander modifications
    ↓
Mise à jour de 02-Brain/ si nécessaire
```
