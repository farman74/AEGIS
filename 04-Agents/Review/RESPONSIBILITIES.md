# Agent Review — Responsabilités

## Responsabilités

1. Vérifier la conformité de chaque PR avec le Feature Contract
2. Vérifier la conformité architecturale
3. Vérifier la qualité du code
4. Vérifier la complétude des tests
5. Vérifier la complétude de la documentation
6. Approuver ou demander des modifications
7. S'assurer que Supabase est utilisé exclusivement
