# Agent Frontend — Responsabilités

## Responsabilités

1. Implémenter les composants React d'ExamOS
2. Implémenter les pages définies dans le Feature Contract
3. Connecter l'interface au backend via le client Supabase officiel
4. Assurer l'expérience correcteur (interface de validation des copies)
5. Gérer les états et les erreurs côté client
6. Écrire les tests de composants

## Ce que cet agent ne fait PAS

- Ne place pas de logique métier dans les composants UI
- N'expose pas les secrets côté client
