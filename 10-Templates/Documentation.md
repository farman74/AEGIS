# Template — Documentation

## Objectif

Fournir le modèle standard pour tout document interne AEGIS.

## Quand utiliser

À chaque création d'un nouveau document dans le dépôt AEGIS.

## Prérequis

- Le document doit être prévu par `nouvellesnormes.pdf` ou `README.md`.

## Procédure

---

```markdown
# [TITRE DU DOCUMENT]

## Objectif

[Décrire en une phrase l'objectif de ce document.]

## Quand utiliser

[Décrire dans quel contexte ce document doit être consulté.]

## Prérequis

- [Prérequis 1]

## Procédure

[Décrire les étapes ou le contenu principal.]

## Validation

[Décrire comment valider que ce document est complet et correct.]

## Références

- [Lien interne 1]
- [Lien interne 2]
```

---

## Validation

Le document est validé après passage de la checklist `09-Checklists/Documentation.md`.
