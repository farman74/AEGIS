# 10-Templates

## Objectif

Fournir des modèles officiels uniformes pour tous les livrables produits dans le projet ExamOS.
Tout document ou livrable produit doit respecter le template officiel correspondant.

## Structure

```
10-Templates/
├── README.md
├── ADR.md
├── API.md
├── Feature.md
├── Prompt.md
├── PR.md
├── Issue.md
├── Test.md
└── Documentation.md
```

## Comment utiliser

1. Identifier le type de livrable à produire.
2. Ouvrir le template correspondant.
3. Copier la structure et remplir chaque section.
4. Ne jamais modifier la structure du template.
